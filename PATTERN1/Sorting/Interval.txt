INTERVAL PROBLEMS 
What Are Intervals? 
An interval = a range with start and end point
[1, 3] means → from 1 to 3
[2, 6] means → from 2 to 6
Types of Interval Relationships 
Case 1 — No overlap:
[1,3]  [5,7]
     ↑↑
     gap exists

Case 2 — Overlap:
[1,4]
   [3,6]
    ↑
    overlap → merge to [1,6]

Case 3 — One inside another:
[1,6]
 [2,4]
  ↑↑
  completely inside → merge to [1,6]

Case 4 — Touch at boundary:
[1,4]
    [4,6]
     ↑
     touching → merge to [1,6]
When Do Two Intervals Overlap? 
intervals A=[a1,a2] and B=[b1,b2] overlap when:
b1 <= a2
↑
start of B is before end of A!


TRIGGER PHRASES 
If problem says...                       Think...
"merge overlapping intervals"            Sort by start + merge
"meeting rooms"                          Sort by start + check overlap
"insert interval"                        Find position + merge
"minimum rooms needed"                   Sort + min heap
