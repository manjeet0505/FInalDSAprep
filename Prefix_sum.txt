What Is Prefix Sum?
Prefix Sum is a precomputation technique.
Instead of recalculating sum of a range every time — you precompute it once and answer any range query in O(1).

The Core Idea
arr    = [2, 4, 1, 3, 5]
index  =  0  1  2  3  4

prefix = [0, 2, 6, 7, 10, 15]
          ↑
          extra 0 at start (makes math clean)

prefix[i] = sum of all elements from index 0 to i-1

The Magic Formula 
Sum of range [L, R] = prefix[R+1] - prefix[L]

Example:
Sum of [1, 3] = prefix[4] - prefix[1]
              = 10 - 2
              = 8 

Verify: arr[1]+arr[2]+arr[3] = 4+1+3 = 8 

Why The Extra 0 At Start?
Without 0:  prefix = [2, 6, 7, 10, 15]
Sum of [0,2] = prefix[2] - prefix[-1] →  index -1!

With 0:     prefix = [0, 2, 6, 7, 10, 15]
Sum of [0,2] = prefix[3] - prefix[0]
             = 7 - 0 = 7 

Building Prefix Array
javaint[] prefix = new int[arr.length + 1];
prefix[0] = 0;
for (int i = 0; i < arr.length; i++) {
    prefix[i + 1] = prefix[i] + arr[i];
}

Real World Analogy 
Odometer in a car records TOTAL distance travelled.

Want distance between city A and city B?
→ odometer[B] - odometer[A]

No need to re-measure the road!
That's exactly prefix sum.
